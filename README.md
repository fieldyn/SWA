# InnovatioX SWA

React + TypeScript remake of the landing page, ready for Azure Static Web Apps.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Azure Static Web Apps

- SPA fallback is configured in `public/staticwebapp.config.json`
- Search crawl assets live in `public/robots.txt` and `public/sitemap.xml`
- The app is fully static and builds to `dist/`
